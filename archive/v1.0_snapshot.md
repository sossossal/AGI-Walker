# Snapshot: AGI-Walker v1.0 (Prototype)

**Date**: Mid 2025
**Focus**: Basic Simulation Connectivity

## Description
The initial proof-of-concept connecting Godot and Python via TCP.

## Key Components
*   **godot_project/**: Basic scenes.
*   **simple_tcp_client.py**: Raw socket communication.
*   **pid_controller.py**: Hardcoded walking gait.

## Limitations
*   Unstable TCP connection.
*   No physics randomization.
*   Rudimentary graphics.
