# AGI-Walker 5分钟演示视频脚本

## 🎬 视频信息

- **时长**: 5 分钟
- **目标**: 展示完整的 Sim-to-Real 工作流
- **受众**: 机器人开发者、研究人员

---

## 📝 分镜脚本

### 开场（0:00 - 0:30）

**画面**: 项目 Logo + 标题动画

**旁白**:
> "厌倦了仿真和真实硬件之间的巨大鸿沟？AGI-Walker 为您提供从仿真训练到硬件部署的完整解决方案。"

**文字**: "AGI-Walker: 完整的 Sim-to-Real 机器人开发工具链"

---

### 第一部分：零件库（0:30 - 1:20）

**画面**: 切换到 VSCode，打开零件库 JSON 文件

**旁白**:
> "从真实硬件开始。我们的零件库包含真实制造商的数据规格。"

**演示操作**:
```python
from godot_robot_env import PartsDatabase

db = PartsDatabase()
motor = db.get_part("dynamixel_xl430_w250")

print(f"扭矩: {motor['specifications']['stall_torque']} N·m")
print(f"价格: ${motor['price_usd']}")
```

**画面**: 显示输出
```
扭矩: 1.0 N·m
价格: $49.9
```

**旁白**:
> "精确的性能参数，可靠的成本估算，让您的设计从一开始就基于现实。"

---

### 第二部分：Godot 仿真（1:20 - 2:30）

**画面**: 切换到 Godot 编辑器

**旁白**:
> "在 Godot 中构建您的机器人。真实的物理引擎，丰富的环境配置。"

**演示操作**:
1. 打开预设场景（双足机器人）
2. 按 F5 运行仿真
3. 演示环境切换（按键 1, 2, 3）
   - 地球重力
   - 月球重力（缓慢飘落）
   - 火星重力
4. 演示材质切换（按键 C, I）
   - 混凝土（高摩擦）
   - 冰面（低摩擦，滑行）

**旁白**:
> "轻松切换重力、地面材质、倾斜角度。域随机化训练，缩小 Sim-to-Real gap。"

---

### 第三部分：RL 训练（2:30 - 3:20）

**画面**: 回到 VSCode，Terminal

**旁白**:
> "使用标准的强化学习库训练您的策略。"

**演示操作**:
```python
from godot_robot_env import GodotRobotEnv
from stable_baselines3 import PPO

env = GodotRobotEnv()
model = PPO("MultiInputPolicy", env, verbose=1)
model.learn(total_timesteps=100000)
model.save("walker_policy")
```

**画面**: 显示训练进度条（加速播放）

**文字叠加**: "Gymnasium 兼容 | Stable-Baselines3 | 标准化接口"

**旁白**:
> "完全兼容 Gymnasium 和 Stable-Baselines3。使用您熟悉的工具。"

---

### 第四部分：硬件部署（3:20 - 4:40）

**画面**: 分屏 - 左侧代码，右侧硬件图示

**旁白**:
> "训练完成？一键部署到真实硬件。"

**演示操作**:
```bash
python examples/deploy_to_hardware.py
```

**画面**: 显示部署流程
1. 导出 ONNX 模型
2. INT8 量化
3. 生成 C 数组

**画面**: 切换到硬件架构图

**旁白**:
> "基于 IMC-22 芯片的边缘智能控制器。RISC-V 处理器加神经加速器，1kHz 控制频率，延迟小于 100 微秒。"

**画面**: 展示 Hive-Reflex 架构图
- CAN 总线拓扑
- 分布式节点
- 反射控制公式

**旁白**:
> "独特的脊髓反射架构。每个关节独立运行神经网络，实现超低延迟响应。"

---

### 第五部分：成本分析（4:40 - 4:55）

**旁白**:

## 🎥 拍摄建议

### 工具
- **录屏**: OBS Studio（免费）
- **编辑**: DaVinci Resolve（免费）或 Adobe Premiere
- **配音**: Audacity（免费）

### 技巧
1. **准备 Demo 环境**
   - 清理桌面
   - 放大字体（易于观看）
   - 禁用通知

2. **录制**
   - 先录视频，后配音
   - 分段录制，便于剪辑
   - 保持鼠标移动流畅

3. **后期**
   - 添加背景音乐（轻音乐）
   - 关键点添加文字注释
   - 开头/结尾添加转场

### 导出设置
- 分辨率: 1080p (1920x1080)
- 帧率: 30fps
- 格式: MP4 (H.264)
- 码率: 5-8 Mbps
