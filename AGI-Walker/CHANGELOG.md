## 发布说明

### v0.9.0-beta

这是 AGI-Walker 项目的首个公开测试版本。我们整合了 Hive-Reflex 分布式反射控制器，创建了业界首个完整的开源 Sim-to-Real 机器人开发工具链。

**核心价值**:
- 从零件选型到硬件部署的完整流程
- 真实硬件规格的数字孪生
- 独特的边缘智能控制架构

**适合谁使用**:
- 机器人研究人员
- RL 算法开发者
- 硬件原型工程师
- 教育工作者

**快速开始**:
```bash
pip install -r requirements.txt
cd examples
python quick_start_balance.py
```
