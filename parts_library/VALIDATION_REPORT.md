## 使用示例

```python
# 创建定制电机
motor = CustomMotor({
    'power': 500,
    'gear_ratio': 50
})

# 查看初始性能
print(motor.get_performance_report())

# 调整参数并查看影响
impact = motor.update_param('gear_ratio', 100)
print(f"影响: {impact['impact']}")

# 性能自动更新
print(f"新扭矩: {motor.performance_metrics['output_torque_nm']:.2f} Nm")
```
